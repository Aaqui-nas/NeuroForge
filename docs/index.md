# NeuroForge

A desktop application for visually designing, training, analyzing and versioning PyTorch neural networks.

Think **Unreal Engine Blueprint** — but for deep learning.

---

## What is NeuroForge?

NeuroForge lets you:

- **Design** neural architectures by connecting nodes in a visual graph editor
- **Train** models locally (or on remote machines) without writing boilerplate
- **Monitor** training in real time — the graph itself comes alive with color-coded gradient and activation information
- **Version** everything — architectures, datasets, checkpoints, experiments — in a Git-inspired object store
- **Compare** runs and fork from any checkpoint

No web browser. No cloud. A native desktop application.
